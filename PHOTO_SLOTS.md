# Photo Slot Reference

Use these codes when replacing product or lifestyle photography from Drive.
If a client labels a photo with one of these codes, place that image into the matching website slot.

## Homepage

- `H-01`: Homepage hero lifestyle image
- `C-01`: Homepage category card, Women / Kadın
- `C-02`: Homepage category card, Men / Erkek
- `C-03`: Homepage category card, Kids / Çocuk
- `C-04`: Homepage category card, Tote Bags / Bez Çanta
- `P-01`: Homepage product card 1
- `P-02`: Homepage product card 2
- `P-03`: Homepage product card 3
- `P-04`: Homepage product card 4
- `L-01`: Homepage large tote lifestyle image
- `L-02`: Homepage small tote lifestyle image
- `G-01` and up: Instagram/gallery grid images, in visual order

## Tote Bag Page

- `D-01`: Page hero/detail image
- `T-01`: Art Collection
- `T-02`: Amsterdam Collection
- `T-03`: Stock Models
- `T-04`: Custom Printed Bags
- `T-05`: Business / Cafe Bags

## Other Detail Pages

- `D-01`: Top hero/detail image for the current page
- `P-01` and up: Product cards on the current page, in visual order

## Working Rule

When importing photos from Drive:

1. If the file name, folder name, or client note contains a slot code, use that code as the placement instruction.
2. Prefer replacing the image for the exact slot instead of changing layout or copy.
3. Keep the subtle visible code on the website after replacement, so future feedback stays easy.
4. If multiple photos are given for the same code, use the most polished lifestyle/product image first and keep the rest as alternates.
